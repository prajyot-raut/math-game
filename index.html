About

The aim of this game is improve your Mathematics Calculation. This game is very useful for students.

Credits Developer : Prajyot Raut Designer : Prajyot Raut

How to Play

  • After starting the game, random Math expression will appear.
  • Just calculate that expression and enter it using the number Button given in the game.
  • Try to calculate as much as expression you can calculate.
  • If the Math expression box touched the red line then your game will be over.
